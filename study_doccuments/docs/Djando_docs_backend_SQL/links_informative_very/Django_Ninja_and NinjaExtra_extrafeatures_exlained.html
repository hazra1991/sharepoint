✅ What is RouteGuard?

🔒 A RouteGuard is like a reusable, class-based permission checker.

It lets you enforce authorization on a per-route or whole-controller level.

🔹 Step 1: Define a guard class

python
from ninja_extra.permissions import RouteGuard
from ninja_extra import APIException

class IsAdminGuard(RouteGuard):
    def has_permission(self, request, view_func):
        auth = request.headers.get("Authorization")
        if auth == "Bearer admin-token":
            return True
        raise APIException(status_code=403, detail="Forbidden")

  • has_permission() is called before the route runs.

  • If it returns False or raises an exception, the request is blocked.

🔹 Step 2: Apply guard to a route

python
from ninja_extra import Controller, http_get

class AdminController(Controller):

    @http_get("/admin", guards=[IsAdminGuard])
    def admin_view(self, request):
        return {"admin": True}

Now /admin is only accessible if header = Bearer admin-token.

🔹 Step 3 (Optional): Apply guard to whole controller

python
class WholeGuardedController(Controller):
    guards = [IsAdminGuard]  # Applies to all endpoints

    @http_get("/dashboard")
    def dashboard(self, request):
        return {"msg": "Welcome Admin"}
